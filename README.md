# Expense Tracker App

A simple Expense Tracker application built using Python and Kivy.

## Features

- Add Expenses
- Delete Expenses
- Previous/Next Navigation
- Total Expense Calculation
- JSON Data Storage
- Mobile Friendly Interface

## Technologies Used

- Python
- Kivy
- JSON

## Files

- main.py
- expenses.json
- README.md

## How to Run

1. Install Kivy
2. Run:

```bash
python main.py
```
